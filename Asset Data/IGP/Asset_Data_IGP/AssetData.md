## ASSET DATA 

This folder contains geospatial data for the following assets:

1) **Brick Kiln**
2) **Cement**
3) **Steel**
4) **Transport**
5) **Open Burning sites**
6) **Coal**
7) **Oil and Gas**
8) **Pulp and Paper Industry**
9) **Chemical Industry**
10) **Crop Stubble Burning** (seasonal)

### Processing Steps

**DATA ACQUISITION**

- Acquire data from open data sources.
- Clip them to IGP or AOI
- Then estimate the pollution emission
- Store as csv and geojson.
